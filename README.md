## Form Builder Application

# Features
- Drag and Drop form builder.
- Customizable form elements.
- Save and review templates.
- Form validations.

# Supported elements
- Label
- Input
- Select Dropdown
- Radio Input
- Checkbox
- Button

# How to run the project

Dependencies
- Node +20 

```
yarn install
```

```
yarn dev
```

To build

```
yarn build
```
